# 문서 작성 규칙

Python 학습 문서는 실습자가 바로 실행하고 확인할 수 있게 작성합니다.

## 기본 형식

- 문서 제목은 `{번호}. {주제}` 형식으로 작성합니다.
- 개념 설명 뒤에는 실행 가능한 예제를 둡니다.
- 마지막에는 `확인 포인트`를 둡니다.
- 긴 설명보다 판단 기준과 실습 흐름을 우선합니다.

## 코드 예제

- 예제 코드는 Python 3.11 이상 기준으로 작성합니다.
- 함수에는 가능한 타입 힌트를 붙입니다.
- 파일 시스템, 네트워크, subprocess 예제는 실패 가능성과 예외 처리를 함께 설명합니다.

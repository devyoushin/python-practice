# Python 코드 규칙

이 저장소의 예제 코드는 학습용이지만, 실무 코드에 가까운 형태를 유지합니다.

## 기본 원칙

- 실행 진입점은 `main()`과 `if __name__ == "__main__":` 패턴을 사용합니다.
- 계산 로직과 입출력 로직을 분리합니다.
- `print()`는 예제 출력에 쓰고, 운영 흐름은 `logging`을 사용합니다.
- 외부 명령 실행은 `subprocess.run([...], check=True)` 형태를 기본으로 합니다.
- 네트워크 요청에는 항상 timeout을 둡니다.

## 타입과 테스트

- 공개 함수에는 타입 힌트를 붙입니다.
- 테스트 가능한 로직은 함수로 분리합니다.
- 예외는 구체적인 타입으로 잡습니다.

# Python Docs

Python 학습 문서는 주제별로 나눠 관리합니다.

| 폴더 | 내용 |
|------|------|
| `install/` | Python 설치, venv, pip, 실행 환경 |
| `basics/` | 변수, 타입, 조건문, 반복문, 함수 |
| `runtime/` | 실행 모델, 스코프, 예외 처리 |
| `data/` | list, dict, set, tuple, 문자열, 날짜 |
| `oop/` | 클래스, 상속, dataclass |
| `modules/` | 모듈, 패키지, import 구조 |
| `io/` | 파일 입출력, JSON, YAML, CSV |
| `testing/` | pytest, fixture, mocking, coverage |
| `automation/` | CLI, argparse, subprocess, 자동화 스크립트 |
| `web/` | HTTP client, API 호출, FastAPI 기초 |
| `operations/` | logging, typing, packaging, 성능 점검 |

처음 읽을 문서는 `install/01_environment.md`입니다.

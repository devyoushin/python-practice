# python-practice 문서

이 디렉토리는 주제별 번호 체계로 문서를 정리합니다. 앞 번호는 학습과 운영 흐름을 나타내며, 90번대는 저장소 관리용 문서입니다.

## 문서 구조

- [01-installation](./01-installation/): Python 환경 구성 - Python 실행 환경과 개발 환경 준비 문서를 모읍니다.
- [02-language](./02-language/): Python 언어 기본 - 변수, 흐름 제어, 함수, 예외, 클래스 문서를 모읍니다.
- [03-stdlib](./03-stdlib/): 표준 라이브러리 - 모듈, 파일 처리, subprocess, logging 문서를 모읍니다.
- [04-tooling](./04-tooling/): 개발 도구 - pytest, typing, packaging 문서를 모읍니다.
- [05-automation](./05-automation/): 자동화 - CLI와 자동화 스크립트 문서를 모읍니다.
- [06-web](./06-web/): 웹 개발 - FastAPI와 HTTP API 기본 문서를 모읍니다.
- [90-standards](./90-standards/): 문서 작성 표준 - 저장소 문서 작성 규칙과 운영 기준을 관리합니다.
- [91-templates](./91-templates/): 문서 템플릿 - 반복해서 사용하는 문서 템플릿을 관리합니다.
- [99-agents](./99-agents/): AI 에이전트 가이드 - Python 코드 리뷰와 문서화에 활용할 에이전트 역할 정의를 관리합니다.
